# AGENTS.md

## Website Design & UI/UX Rules

When working on this website, act as both a senior frontend developer and a UI/UX designer.

Always prioritize:
- Clean visual hierarchy
- Mobile-first responsive design
- Professional spacing
- Strong hero section
- Clear call-to-action buttons
- Consistent typography
- Brand-aligned colours
- Accessibility and readable contrast
- Fast-loading pages
- Production-ready code

## Design Direction

The website should feel:
- Trustworthy
- Modern
- Local and authentic
- Professional
- Easy for customers to understand within 5 seconds

Avoid:
- Generic template designs
- Crowded layouts
- Random colours
- Rebuilding the entire project unless asked
- Adding unnecessary libraries

## Coding Rules

- Preserve the existing project structure.
- Use Tailwind CSS if the project already uses Tailwind.
- Keep components reusable.
- Do not change routing unless requested.
- Do not remove existing content unless instructed.
- Make incremental improvements.

## Final Review Checklist

Before finishing, check:
1. Does the page look good on mobile?
2. Is the main CTA obvious?
3. Is the spacing clean?
4. Is the text readable?
5. Does the design match the business?
6. Does the site feel professional enough for a paying client?
